# Self-Assessment: Ist mein Use Case ein AI Use Case?

Diese kleine Single-Page-App hilft dir per Self-Assessment einzuschätzen, ob dein Anwendungsfall (Use Case) ein Kandidat für eine KI-basierte Lösung ist.

## Was die App macht
- Stellt gewichtete Ja/Nein‑Fragen in verständlicher Sprache.
- Berechnet eine Punktzahl und klassifiziert den Use Case (Wahrscheinlichkeit).
- Gibt eine transparente Erklärung und konkrete nächste Schritte (Daten‑Checkliste, MVP‑Ideen).
- Export der Antworten als JSON möglich.

## Dateien
- index.html — Hauptseite
- assets/style.css — Styles
- assets/app.js — Logik und Fragen

## Lokal starten
Einfach `index.html` im Browser öffnen. Keine Installation nötig.

## Vorschläge zur Weiterentwicklung
- Persistenz (Speichern der Assessments in einem Backend oder GitHub Gist).
- Branching‑Logik (Folgefragen basierend auf Antworten).
- Automatische Vorschläge für ML‑Modelle / Libraries basierend auf Antworten.
- Formular‑Sharing (Link mit Vorkonfigurierten Antworten).
